# Практика

Материалы для практической работы и самопроверки.

## Содержание

- [**Практическое задание: распределённая система оплаты услуг**](practice.md) — большой сквозной проект, охватывающий ООП, HTTP, БД, GUI, логирование и тестирование.
- [**Шпаргалка**](cheatsheet.md) — краткий справочник по типам Python и области видимости (с параллелями в Go).
- [**Системы контроля версий и Git**](version-control.md) — обзор Git, базовые команды, ветвление, Conventional Commits, современные инструменты.
- [**Курсовые лабораторные**](../labs/index.md) — список лабораторных работ по темам курса (объём в часах, что сдавать и как). Это учебные задания, привязанные к лекциям, — не путать с [DSA-лабами](../notes.md) `lab00`–`lab11`, которые реализованы в репозитории.
- [**Экзаменационные билеты**](../tickets/index.md) — 26 билетов для самопроверки: теория по темам курса, блок-схема и реализация алгоритма.
