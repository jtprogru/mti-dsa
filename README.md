# mti-algos

Лабораторные работы по предмету «Алгоритмы и структуры данных».

## Требования

- Python >= 3.14
- [uv](https://docs.astral.sh/uv/) для управления зависимостями

## Установка

```bash
make sync
```

## Запуск лабораторных

```bash
make run                 # список доступных лабораторных
make run lab01           # запустить конкретную лабораторную
make run lab02_random
```

## Тесты

```bash
make test
```

## Прочие команды

```bash
make help                # список всех команд
make clean               # удалить кэш и временные файлы
```

## Структура

```
labs/
  lab01.py          # массив и стек на его основе (без встроенных len/range)
  lab02.py          # связный список, очередь и бинарное дерево поиска
  lab02_random.py   # вариант lab02 со случайными данными и сбалансированным деревом
tests/              # тесты к каждой лабораторной
docs/
  notes.md          # общий индекс конспекта со сводной таблицей сложности
  lab01.md          # конспект по lab01 (массив и стек)
  lab02.md          # конспект по lab02 (список, очередь, дерево)
  lab02_random.md   # конспект по lab02_random (случайные данные и баланс)
```

## Конспект

Подробный разбор всех сущностей из лабораторных с примерами кода и оценкой
сложности разнесён по файлам в [docs/](docs/):

- [docs/notes.md](docs/notes.md) — общий индекс, сводная таблица сложности и выводы.
- [docs/lab01.md](docs/lab01.md) — массив и стек.
- [docs/lab02.md](docs/lab02.md) — связный список, очередь, бинарное дерево поиска.
- [docs/lab02_random.md](docs/lab02_random.md) — случайные данные и сбалансированное дерево.

## Лабораторные

### lab01 — массивы и стек

Целочисленный массив со случайным заполнением, функции вывода по индексу, поиска максимума (по значению и индексу) и суммы элементов без встроенных функций, а также стек на основе массива.

### lab02 — связный список, очередь, бинарное дерево

Односвязный список с добавлением/удалением/печатью, очередь (FIFO) на его основе и бинарное дерево поиска. `print_tree` рисует дерево сверху вниз с ветками `/` и `\`.

### lab02_random — случайные данные и сбалансированное дерево

Список и очередь заполняются случайными значениями. Метод `build_balanced` строит дерево из отсортированного массива «серединой вперёд», поэтому его форма не зависит от порядка ввода. Выводятся два дерева: до сортировки (форма зависит от ввода) и после (сбалансированное).
